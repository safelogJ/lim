# Lim (Local IP Messenger)

Автономный мессенджер для Mikrotik CHR. Проект предназначен для тех, кто хочет иметь полностью приватную среду общения внутри своей локальной сети без зависимости от сторонних сервисов и интернета.

### Ключевые особенности:
* **Автономность:** Все запросы и передача данных происходят только по локальным IP-адресам. Подключение к интернету не требуется.
* **Приватность:** Серверная часть устанавливается администратором на собственный Mikrotik CHR (в Docker-контейнере).
* **Личные чаты:** Поддержка строго индивидуальных переписок (1-на-1). Для начала общения необходимо знать уникальный логин собеседника, зарегистрированного на вашем сервере.
* **Передача файлов:** Возможность отправлять текстовые сообщения и файлы.
* **Умное хранение:** Отправленные файлы автоматически удаляются с сервера сразу после того, как получатель успешно их скачал. Это позволяет не переполнять дисковое пространство роутера.
